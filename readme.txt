== Installation ==
1 - Upload the folder to the /wp-content/plugins/ directory
2- Activate the plugin through the ‘Plugins’ menu in WordPress
3- Install the AMP Required plugin
4- You may need to refresh your permalinks by going to Settings > Permalinks and tapping the Save button.
5- Enjoy!
